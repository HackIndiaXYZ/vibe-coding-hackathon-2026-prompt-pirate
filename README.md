# ProductJudge AI

ProductJudge AI is an AI-powered review board that evaluates:

- Startup Ideas
- Hackathon Projects
- Investor Pitches
- Websites
- Mobile Apps
- Product Workflows

The platform simulates a panel of AI experts:

- User Agent
- Investor Agent
- Designer Agent
- Engineer Agent
- Growth Agent
- Judge Agent

Each review generates:

- Scores
- Strengths
- Weaknesses
- Recommendations
- Risk Analysis
- Improvement Roadmap
- Final Verdict
